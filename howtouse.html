Morse Code Cryptographer

How to Use

1. Type your message in the input box.
2. Click "To Morse" to convert text to Morse code.
3. Click "To Text" to decode Morse code back to text.
4. The converted message will appear in their respective output box.
5. You can copy the output from the output box.
6. Enjoy using the Morse Code Cryptographer!

You can utilize the buttons such as play, pause, and stop to control the audio playback of the Morse code.
1. The "Play" button will give the audio of the output message.
2. The "Pause" button will pause the audio.
3. The "Stop" button will stop the playback.
